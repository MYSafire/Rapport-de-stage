Développement :

Pour mon rapport de stage j'ai décidé de le former par un site web avec mon formateur prénomer Julien . Un site web est former par des pages web qui sont elles mêmes composées par des languages de codes . Les languges sont :

- HTML : Langage de balisage utilisé pour la création de pages web, permettant notamment de définir des liens hypertextes.

- CSS : Le CSS est un langage informatique utilisé sur l'internet pour mettre en forme les fichiers HTML ou XML. Ainsi, les feuilles de style, aussi appelé les fichiers CSS, comprennent du code qui permet de gérer le design d'une page en HTML.

- JAVA SCRIPT : Le JavaScript est un langage informatique utilisé sur les pages web. Ce langage à la particularité de s'activer sur le poste client, en d'autres mots c'est votre ordinateur qui va recevoir le code et qui devra l'exécuter. C'est en opposition à d'autres langages qui sont activé côté serveur.

Ces languages ont des balises, certaines importantes comme les balises " head " et " body " utilisé uniquement pour le language HTML .

Pour le language CSS les balises les plus importantes sont " color " et " background " utilisé uniquement pour le language CSS.

Et pour le language JAVA SCRIPT il n'y a pas de balises car c'est un languge de progammation pour animer les pages .
